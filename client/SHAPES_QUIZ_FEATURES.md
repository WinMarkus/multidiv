# Shapes & Geometry Quiz - New Features! 🎮

## Visual Shape Identification

The Shapes Quiz now includes **interactive visual learning** with two types of shape displays:

### 2D Shapes (SVG Graphics) 👁️
- Beautiful, colorful SVG renderings of 2D shapes
- Includes: Triangle, Square, Rectangle, Circle, Pentagon, Hexagon, Octagon, Rhombus, Trapezoid
- Random colors for each question to keep it engaging
- Kids identify shapes by sight, not just by description

### 3D Shapes (Interactive Three.js) 🎮
- **Fully interactive 3D models** that kids can rotate with their mouse or touch!
- Drag to spin the shape and see it from all angles
- Auto-rotates gently when not being dragged
- Includes: Cube, Sphere, Cylinder, Cone, Pyramid
- Works on both desktop (mouse drag) and mobile (touch and drag)
- Helps develop spatial reasoning and 3D visualization skills

## Question Types

1. **Visual 2D Identification** - "What shape is this?" with SVG display
2. **Interactive 3D Identification** - "What 3D shape is this? (Drag to rotate!)"
3. **Text-based 2D Questions** - "How many sides does a hexagon have?"
4. **Text-based 3D Questions** - "How many faces does a cube have?"
5. **Angles** - "How many degrees in a right angle?"
6. **Geometry Concepts** - "Lines that never meet are called..."

## Technical Implementation

- **Three.js** for 3D rendering and interaction
- **SVG** for crisp, scalable 2D shapes
- Mouse and touch event handling for cross-platform support
- Proper cleanup and memory management for 3D scenes
- Responsive design that works on all screen sizes

## Educational Benefits

- **Visual Learning**: Kids learn to recognize shapes by sight
- **Spatial Reasoning**: Rotating 3D shapes helps understand geometry in space
- **Engagement**: Interactive elements make learning fun and memorable
- **Multi-sensory**: Combines visual, kinesthetic (dragging), and cognitive learning

Enjoy exploring shapes! 🧊📐✨
